*************************************Application Information*************************************

This application helps user to make their list of their day to day lives or making any shopping list.

Initially the list will be empty. Once you save your events they will keep on saving in your to dolist table which will stay even on page refresh or new session opening with the same localhost id of the page unless you delete it as the data is stored in your local storage.

you can enter the task and save it using the Save Button.

once saved the data will show in the to do list table underneath.

the data can be marked completed using the checkbox provied.
the task can also be deleted from the list if no longer needed. In case of deletion the data will not show again unless the same entry is made again to the list using the save button.
